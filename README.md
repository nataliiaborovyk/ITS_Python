# ITS_Python – Esercizi di Programmazione Python (ITS ICT Academy)

Questo repository contiene esercizi sviluppati durante lo studio della **programmazione Python** nel percorso ITS ICT Academy.

Include esempi ed esercizi relativi a:

- fondamenti del linguaggio Python
- programmazione orientata agli oggetti
- piccoli script e programmi di esercitazione
- primi esempi di utilizzo del framework Flask

Il repository rappresenta il percorso di apprendimento della programmazione Python durante il corso.

Questo repository fa parte del mio percorso di apprendimento durante il programma Data Analyst presso ITS ICT Academy.
